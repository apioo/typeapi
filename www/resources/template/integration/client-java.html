To integrate the generated client SDK you need to add the app.sdkgen.client package to your project. Then you can simply paste the generate code into the source folder and start using the client for example:

Anonymous credentials = new Anonymous();
Client client = new Client("http://127.0.0.1:1080", credentials);

HelloWorld message = client.test().getHello();

System.out.println("Message: " + message.getMessage());
